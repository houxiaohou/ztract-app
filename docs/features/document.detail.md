# 文档 document 抽取结果查看页面

此文档描述了 document 抽取成功后点击查看详情后的页面。

## 用户操作流程
1. 在 Documents 页面（/projects/<pid>/documents），针对 Status = Success 的 document（也就是抽取成功后的文档）点击文档名字进入结果查看页面；
2. 此页面的路由可以是：/projects/<pid>/documents/<did>
3. 此页面展示界面分为两栏：左侧为图片预览文档、右侧是抽取的数据结果；
4. 左侧图片文档预览又分为两栏，最左侧有个 page 的缩略图，右侧为大图；
5. 抽取的数据需要根据 @xparse-kit/visualizer 这个开源项目在图片上标出所在的位置，也就是框选出来，点击可以联动右侧的抽取结果高亮显示。

## 页面布局
document 抽取详情页面参考图片：`@docs/features/document.detail.png`

## 用户上传文档缩略图
- TextIn 抽取的数据结果中返回了文档这一页的预览图，你需要根据后端提供的接口来读取这个预览图的地址进行展示。
