# Parsed Data 相关需求说明

项目详情中有一个 Parsed Data 页面，此页面用于展示已经抽取的全部数据。
项目详情中增加一个导出记录页面，此页面用于展示所有的导出记录，支持下载、删除。

## Parsed Data 页面布局、用户操作、交互需求
- 页面布局：
  - 顶部展示操作栏：包含日期筛选、导出按钮）
  - 中部 table 布局展示已抽取的数据：table 的 header 是 project 的 schema
  - 最底部是翻页按钮
- 顶部筛选
  - 可以按照抽取日期范围进行筛选
- 顶部导出
  - 点击导出按钮，弹窗显示，用户选择导出的格式（支持 xlsx、csv、json）、筛选日期（默认全部）
  - 确认导出后，toast 提示提交导出任务成功，后续前往导出记录查看

## 导出记录页面布局
- 做项目详情左侧导航栏增加一个导出记录的路由
- 展示全部历史导出，支持翻页
- 对于已经 success 的导出记录，可以下载
- 支持删除单条导出记录
