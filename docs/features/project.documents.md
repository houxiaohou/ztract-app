# 项目文档页面需求

此文档描述了项目想请下，文档页面的具体需求。

## 页面核心功能

- 顶部搜索框，可以输入文档的名字进行搜索
- 搜索框右侧有个上传文档按钮
- 点击上传按钮后在顶部搜索框下面展开一个文件上传区域，支持拖拽、选择上传，支持多文件上传，最大支持 50M，支持 word、excel、ppt、html、txt、image
- 上传区域下方是项目文档列表，表格列出已经上传的文档，表格字段包含：状态、文件名、上传时间、解析时间、文件页数、额度消耗
- 如果接口提供的字段不满足前端需求，你最后再告诉我，我去调整后端接口。
- 项目文档支持翻页
- 如果项目下面还没有上传文档，文件上传区域默认展开

## 文件上传逻辑
- 使用 cloudflare R2 上传文件，通过接口获取上传链接，前端直传 R2
- 上传成功后，调用接口将 Document 添加到 Project 下面，api.json 文档里面有相关接口
