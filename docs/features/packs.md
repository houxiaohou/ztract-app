# 套餐包购买、订单页面

此文档描述了付费套餐包的购买页面需求、以及历史订单页面需求。

## 套餐包购买页面

此页面对应 layout 导航栏上可用额度点击后的跳转页面。

### 套餐包逻辑

ztract 实行 pay-as-you-go 模式，用户无需按月订阅付费，只需要选择套餐包下单购买，套餐包额度有效期 1 年。

### 套餐包配置

参考 `@docs/api.json` 中的相关接口，可以获取可用的套餐包。

### 套餐包展示

使用常见的 saas 网站定价卡片展示，卡片 展示名字、价格、有效期、购买按钮。

### 套餐包购买

点击购买后，参考提供的 api 文档，跳转 stripe 进行购买支付。


## 订单页面

此页面对应用户购买的历史订单。

### 订单展示方式

- 使用列表展示历史订单，每个项目是一个购买成功的订单，信息包含：购买时间、购买套餐、购买额度、价格、以及获取 invoice 按钮
- 需要有翻页功能。
