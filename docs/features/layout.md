# 应用布局

此文件定义了整个应用的布局结构。 除了 /auth 开头的路由、404 页面外，所有的 App 页面都需要应用此布局。

## 顶部导航栏

- 顶部导航栏是整个 app 共用，除了上述的页面外；
- 顶部导航栏最左侧是产品 logo；
- 导航栏最右侧是用户的头像，圆形，背景色是主题色，默认用用户昵称的第一个字母或者邮箱的第一个字母填充；
- 头像左侧是剩余额度，数据获取请参考 `@docs/api.json` 文件中定义的 /quota/me 接口，额度形式为⚡️的 icon + 数字，点击后跳转至套餐包购买页面。


### 头像下拉菜单
点击头像是个下拉菜单，下拉菜单从上大小包括：
- 昵称（如果有）
- 邮箱
- Settings
- Bills
- 退出登录

## 主要内容区域

顶部导航栏下侧就是主内容区域，首页目前留空，后续再补充。

## 你的任务

根据此文件的描述完成应用布局的代码，需要结合接口文件进行数据请求，填充真实数据。
