# 项目详情页导航栏需求描述

此项目描述了项目详情页的相关页面布局、功能点。

## 页面布局

- 此页面不继承 App 的布局，需要有个独立的布局 Layout。
- 布局方式为：左侧菜单栏、右侧功能区；
- 右侧功能区顶部为 Header 部分，Header 右侧与目前的顶部导航栏布局相同，额度、个人头像；Header 左侧为项目名称。

## 左侧菜单栏

左侧是项目详情的菜单栏，包含顶部项目信息展示、中部的导航栏。

顶部项目信息展示：
- 展示目前选中的项目名称、描述
- 有返回首页的后腿按钮

中部的导航栏有几下几个导航项：
1. Documents
2. Parsed Data
3. Extraction Schema
导航项选中后，切换到指定的子路由，菜单栏的此项为选中状态；所有的子路由页面共享项目详情布局。


## 你的任务
- 撰写符合上述需求的代码，包括布局、子路由页面。
- 子路由页面先留空，后续我再要求你补充；
- 需要对接后端接口的项目详情接口，因为url路由传入项目 id，据此进行项目详情的获取。
